Кроки реалізації

  1. GitHub репозиторій — створюєш репо, вмикаєш GitHub Pages
  2. Decap CMS — додаєш папку admin/ з index.html та config.yml
  3. GitHub OAuth App — реєструєш застосунок для авторизації
  4. Cloudflare Workers — деплоїш OAuth проксі з Client ID та Secret
  5. config.yml — налаштовуєш колекцію товарів з потрібними полями
  6. Admin панель — заходиш на /admin, логінишся через GitHub
  7. Додаєш товари — CMS комітить JSON + фото в репозиторій
  8. Фронтенд — JavaScript завантажує товари через GitHub API і відображає на сайті
